Classe pour le traitement des données.

class Data_class.Data(fichiers_ids, output_fichier='output.csv')[source]

Bases : object

Classe de traitement des données. Permet de télécharger, fusionner et sauvegarder des fichiers de données (CSV, JSON, GeoJSON) depuis Google Drive.

Paramètres:

  • fichiers_ids – Dictionnaire des ID de fichiers Google Drive, où la clé est le nom du fichier et la valeur est l’ID Google Drive.

  • output_fichier – Nom du fichier de sortie pour les données fusionnées. Par défaut « output.csv ».

telechargement_bd_drive(fichier_id)[source]

Télécharge un fichier depuis Google Drive en utilisant son ID et le charge directement en mémoire. La méthode détecte le type de fichier (CSV, JSON, GEOJSON) et le charge en conséquence.

Paramètres:

fichier_id – L’ID du fichier à télécharger depuis Google Drive.

Renvoie:

Le contenu du fichier téléchargé sous forme binaire.

charger_fichier(fichier_content, fichier_nom)[source]

Charge un fichier en DataFrame ou GeoDataFrame en fonction du type de fichier. Les fichiers sont traités en mémoire sans être enregistrés localement.

Paramètres:

  • fichier_content – Le contenu du fichier téléchargé (binaire).

  • fichier_nom – Le nom du fichier, utilisé pour déterminer son type (CSV, JSON, GeoJSON).

Renvoie:

Un DataFrame ou GeoDataFrame si le fichier est valide, sinon None.

fusionner_fichiers()[source]

Fusionne tous les fichiers téléchargés depuis Google Drive en un seul DataFrame.

Renvoie:

Un DataFrame fusionné de tous les fichiers téléchargés, ou None si aucun fichier valide.

sauvegarder_csv(df)[source]

Sauvegarde le DataFrame fusionné dans un fichier CSV localement.

Paramètres:

df – Le DataFrame fusionné à sauvegarder.

executer()[source]

Cette méthode coordonne l’exécution de toutes les actions de traitement des données.

Elle télécharge, fusionne et sauvegarde les fichiers traités en un seul CSV.
