Documentation API

CirculationDuJour.get_color(intensity)[source]

Détermine une couleur en fonction de l’intensité donnée.

Paramètres:

param (int) – La valeur numérique représentant l’intensité doit être un entier positif.

Renvoie:

Une chaîne de caractères représentant une couleur : - “darkred” : si l’intensité est supérieure à 2000, - “red” : si l’intensité est supérieure à 1000, - “darkorange” : si l’intensité est supérieure à 500, - “gold” : si l’intensité est supérieure à 250, - “green” : si l’intensité est inférieure ou égale à 250.

Type renvoyé:

str

fusion_archives.fusion_csv(folder_input, output_file)[source]

Fusionne tous les fichiers CSV d’un dossier donné en un seul fichier CSV de sortie.

Paramètres:

param (str) – Chemin du dossier contenant les fichiers CSV à fusionner.

Renvoie:

Chemin complet du fichier CSV de sortie (incluant le nom et l’extension .csv).

Type renvoyé:

(str)

station_map.add_marker(row)[source]

Ajoute un marqueur sur une carte Folium à partir des données géographiques des stations VéloMagg.

Paramètres:

  • param (pandas.Series) – Une ligne d’un DataFrame contenant les informations nécessaires pour créer un marqueur. La ligne doit inclure au moins les colonnes suivantes :

  • 'latitude' (-) – Latitude de la station.

  • 'longitude' (-) – Longitude de la station.

  • 'nom' (-) – Nom de la station.

  • 'secteur' (-) – Secteur géographique de la station.

  • 'type_stati' (-) – Type de station (exemple : « Vélo » ou « Parking »).

Note: Cette fonction ne peut être utilisé que si un objet map de type folium.Map est déjà créé et accessible.
