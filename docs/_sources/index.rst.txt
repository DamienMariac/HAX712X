Documentation BikeMap
=====================

Bienvenue dans la documentation de BikeMap !

Le projet **BikeMap** a pour objectif d’analyser les flux cyclistes dans la ville à partir de données issues des trajets VéloMagg, des comptages cyclistes, ainsi que des informations cartographiques d’OpenStreetMap.

Dans cette documentation, vous trouverez des informations détaillées sur les modules du projet.
