Packages 
========

Packages pour le traitement des données
---------------------------------------

- **numpy**  

  Cette bibliothèque est fondamentale pour le calcul scientifique avec Python. Elle permet notamment de manipuler des fonctions mathématiques et des tableaux.

- **pandas**  

  Cette bibliothèque est indispensable pour la manipulation et l'analyse de données avec des fonctionnalités de nettoyage.

- **pooch**  

  Cette bibliothèque gère les téléchargements de fichiers de données en vérifiant que les fichiers sont téléchargés et mis en cache de manière efficace.

Packages pour les données géographiques
---------------------------------------

- **geopandas** 

  Extension de *pandas* permettant d'utiliser et de manipuler des données géographiques.

- **shapely**  

  Cette bibliothèque est utilisée pour manipuler et analyser des objets géométriques comme des points, des lignes et des polygones.

- **osmnx**  

  Cette bibliothèque télécharge, modélise et analyse des données géographiques issues d'OpenStreetMap.

- **folium**  

  Cette bibliothèque permet de visualiser des données géospatiales sur des cartes interactives.

Packages pour le web et l'interaction
-------------------------------------

- **beautifulsoup4**  

  Cette bibliothèque permet d'extraire des données de fichiers HTML et XML, facilitant ainsi le web scraping.

- **requests**  

  Cette bibliothèque envoie des requêtes HTTP et est utilisée pour interagir avec des API ou récupérer des données sur le web.

- **selenium** 

  Cette bibliothèque automatise la navigation sur des sites web. Elle permet de simuler des actions utilisateur comme le clic ou la saisie de texte par exemple.

Packages pour la visualisation
------------------------------

- **matplotlib**  

  Cette bibliothèque est utile pour la création de visualisations statiques, animées ou interactives.

- **panel**  

  Cette bibliothèque permet la création d'interfaces utilisateur interactives, comme des tableaux de bord ou des applications web.

Packages pour l'édition multimédia
----------------------------------

- **moviepy** 

  Cette bibliothèque est dédiée à l'édition vidéo. Elle contribue à découper, assembler et manipuler des vidéos.

Packages internes au projet
---------------------------

- **DataProcessing**

  Ce package est destiné à la création et gestion des cartes.

- **MapCrea**

  Ce package contient des modules pour la création et la visualisation de cartes interactives à partir de données géospatiales(flux de circulation, coordonnées géographiques).

Note :
Ces packages et leurs versions sont spécifiés dans le fichier *`requirements.txt`*. Ils doivent être installés pour faire fonctionner ce projet.
